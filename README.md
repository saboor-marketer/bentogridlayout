# Bento Grid Layout

A responsive bento grid layout built with HTML and CSS Grid.

## Features

- **Responsive Design**: Adapts seamlessly from mobile (2-column) to tablet/desktop (4-column) layouts
- **CSS Grid**: Uses modern CSS Grid for flexible and powerful layout control
- **Scalable Typography**: Font sizes increase as screen size grows for optimal readability
- **Hover Effects**: Smooth transitions and hover animations for better UX
- **Modern UI**: Glassmorphism effects with gradient backgrounds

## Responsive Breakpoints

- **Mobile (< 768px)**: 2-column grid
- **Tablet (768px - 1024px)**: 4-column grid with item spanning
- **Desktop (> 1024px)**: Enhanced spacing and larger typography

## How to Use

1. Open `index.html` in your web browser
2. Resize your browser window to see the responsive behavior
3. Hover over items to see the interactive effects

## Customization

- Modify colors in `styles.css` (gradients and backgrounds)
- Adjust breakpoints in media queries
- Change grid spans for different item arrangements
- Update content in `index.html`

## Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Media Queries, Transitions)
